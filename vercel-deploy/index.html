GeoAgent

App de campo para registro geológico.
Solo para Android.

⬇️ Descargar APK

Cómo instalar

1

Descargá el APK tocando el botón de arriba.

2

Abrí Ajustes → Seguridad y habilitá "Instalar apps desconocidas" para tu navegador.

3

Abrí el archivo descargado desde las notificaciones o el administrador de archivos.

4

Tocá Instalar y abrí GeoAgent.
